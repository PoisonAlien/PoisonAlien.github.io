Projects
maftools
PoisonAlien/maftools
maftools is an R package for analyzing data from large-scale cancer genome projects. What started as a fun project while trying to learn R - ended up in a super-productive package which is being downloaded widely from Bioconductor repository.
methrix
CompEpigen/methrix
Methrix is an R package for fast and flexible DNA methylation analysis. Bedgraph files generated by bisulfite pipelines often come in various flavors. Critical downstream step requires aggregation of these files into methylation/coverage matrices. This step of data aggregation is done by Methrix, including many other useful downstream functions.
TCGAmutations
PoisonAlien/TCGAmutations
Ever tried working with TCGA mutation datasets? Its a real pain in the posterior. Well, this package tries to mitigate the issue by providing pre-compiled, curated somatic mutations from 33 TCGA cohorts along with relevant clinical information for all sequenced samples (from Broad Firehose and TCGA MC3 Project)
chiptk
PoisonAlien/chiptk
chiptk is a set of optimized protocols for 50-bp ChIP-seq datasets. Performs read-alignment, peak calling with MACS2 and fast Super Enhancer identification via bwtool.
peakseason
PoisonAlien/peakseason
An R package for rapid bigWig file summarization/analysis.
gtfsanta
PoisonAlien/gtfsanta
gtfsanta is a stand-alone gtf parser written in Julia. It works out of the box with no dependecies.
